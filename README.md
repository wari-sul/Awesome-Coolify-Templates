# lobechat-db-coolify-template
What you must set in Coolify UI
DB_URL_LOGTO → postgres://<SERVICE_USER_POSTGRES>:<SERVICE_PASSWORD_POSTGRES>@postgres:5432/logto.​

DATABASE_URL_LOBECHAT → postgres://<SERVICE_USER_POSTGRES>:<SERVICE_PASSWORD_POSTGRES>@postgres:5432/lobechat.​

LOGTO_ENDPOINT, LOGTO_ADMIN_ENDPOINT, LOBECHAT_PUBLIC_URL, OPENAI_API_KEY as actual HTTPS URLs and key, no trailing slash on LOBECHAT_PUBLIC_URL.
